<注意事項>
このプログラムのリバースエンジニアリング、
    無許可な紹介、再配布、またはそれに類する事はできません。
また、このプログラムを使用し、
    いかなる被害を被ったとしても、その責任は一切負いません。

<紹介>
独自のインターフェースを持つテキストエディタです。
テキストファイル専用でバイナリファイルは開けません。
ANSI、UNICODE、BigEndianUNICODEに対応しています。
関連付けなどによる起動にも対応しています。

<ショートカット機能>
CTRL+N: 新規
CTRL+O: 開く
CTRL+S: 上書き保存
CTRL+A: すべて選択
CTRL+C: 選択範囲のコピー
CTRL+V: 貼り付け
CTRL+X: 選択範囲のカット
CTRL+Z: 元に戻す(直前1回のみ)
CTRL+F: 検索
CTRL+H: 置換
F3:     選択されている文字列と同じものをそれより下の範囲で検索

<その多機能説明>
ボタンANSI/UNICODE/BigEndianUNICODEを
	切り替えることで文字コードを切り替えることができます。
改行コード変換ボタンで
	改行コードをすべてWindowsの改行コード(CRLF)に変換できます。
右下のMOVEコントロールを
	ドラッグすることで、ウィンドウを移動することができます。
その他、各ボタンにより各種機能を実行できます。

<動作環境>
Windows95/NT以降必須
StretchBltのサポート(たいていの場合OKです)
